# Cookie Clicker

A simple single-page web application featuring a clickable cookie in the center and a counter that tracks the number of clicks.

## How to use

1. Open `index.html` in any modern web browser.
2. Click on the cookie 🍪 to increase the counter.

## Features

- Centered clickable cookie emoji.
- Real-time click counter.
- Visual feedback (scaling) when clicking the cookie.
- **Persistence**: Your click count is saved in a browser cookie and will persist for 1 year, even if you close the browser.
